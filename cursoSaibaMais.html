Java online

No curso de Programação Java as aulas são onlines, através do Microsoft Teams, com turmas de no máximo 10 alunos. As aulas são de segunda a sexta-feiras das 20 às 22 horas. Cada aula tem 2 horas de duração. A duração total do curso vai depender do empenho do aluno, mas pode ser de 3 a 8 messes. Veja abaixo os pré requisitos e o cronograma do curso de Programação Java:

Pré requisitos

Antes de realizar o curso o aluno deve prestar atenção aos pré requisitos exigidos para se iniciar o curso.

  • o aluno deve te noções básicas de programação (usando qualquer linguagem).
  • o aluno deve saber como funciona as condicionais: if...else, while, switch, etc.
  • o aluno NÃO precisa ter conhecimentos de Java.
